## 📖 使用说明

### 获取每日60秒图片

- **方法一**: 使用 `pyinstaller` 编译 `60s.py`，生成可执行文件以获取每日60秒图片，并在本地打开以减少加载时间。🖼️
- **方法二**: 直接在HTML中嵌入API链接，实时获取并显示图片。🌐

### 快捷方式与自动执行

- **快捷方式**: 运行 `1.bat` 和 `2.bat` 脚本，将在桌面创建快捷方式，运行后自动删除。🗂️
- **安装包**: 将工具打包为安装包，支持自动执行和快速部署。📦
