## 📖 使用说明

### 获取每日60s图片

- 使用 `pyinstaller` 编译 `60s.py` 以获取每日60s图片，并在本地打开以减少加载时间。🖼️
- 也可以直接在 HTML 中获取图片。🌐

### 快捷方式与自动执行

- `1.bat` 和 `2.bat` 会在桌面创建快捷方式，运行后会自动删除。🗂️
- 制成安装包后可自动执行。📦
